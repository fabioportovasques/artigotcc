Melhoria do Sistema de Gestão de Veículos (IVMS)
ingles: Improvement of Vehicle Management System (IVMS)


Este trabalho de pesquisa para o desenvolvimento do aplicativo de software do Sistema de Gerenciamento de Veículos (VMS) para Unihomes. O desenvolvimento deste sistema de gestão de associados foi proposto devido às práticas convencionais de gestão manual de veículos da Unihomes, que causaram diversas complexidades na gestão dos veículos, aluguel e detalhes da operação. Principalmente, os problemas reconhecidos são veículos e detalhes de aluguel não estão em um banco de dados, longo tempo de processamento para produzir relatórios, veículos errados estão sendo reservados. Assim, os objetivos centrais deste projeto de desenvolvimento foram resolver a maioria desses desafios e complexidades encontrados devido às práticas convencionais de gerenciamento manual de veículos. Em geral, este projeto de desenvolvimento é uma implementação de um sistema informatizado de gerenciamento de membros que adota o conceito de banco de dados inter-relacionado para armazenar, gerenciar e organizar dados de maneira sistemática e eficiente. Microsoft Visual Studio 2016, Crystal Reports e MYsql foram usados ​​como ferramentas de desenvolvimento chave para este projeto de desenvolvimento. Além disso, a metodologia Agile Unified Process que se estabeleceu benéfica por várias organizações foi adotada e aplicada para todo este projeto de desenvolvimento. Essa implementação garantirá que a usabilidade do aplicativo será fácil, pois o usuário tinha dificuldades para analisar os dados quando o registro era feito manualmente e era difícil para o registro de forma rápida e eficiente.
